Pada praktikum RPL 2 pertemuan 4, saya mempelajari konsep dasar Object Relational Mapping (ORM), yaitu teknik menghubungkan objek dalam program dengan tabel pada database. 
Saya juga belajar menggunakan Hibernate sebagai framework ORM untuk mempermudah pemetaan data tanpa harus menulis query SQL secara manual, sehingga proses pengembangan aplikasi menjadi lebih cepat dan efisien.
